1 IF класс IS голосеменные AND структура листа IS чешуеобразная THEN семейство IS кипарисовые.

2 IF класс IS голосеменные AND структура листа IS иглоподобная AND конфигурация IS хаотическая THEN семейство IS сосновые.

3 IF класс IS голосеменные AND структура листа IS иглоподобная AND конфигурация IS 2 ровных ряда AND серебристая полоса IS да THEN семейство IS еловые.

4 IF класс IS голосеменные AND структура листа IS иглоподобная AND конфигурация IS 2 ровных ряда AND серебристая полоса IS нет THEN семейство IS болотный кипарис.

5 IF type IS деревья AND форма листа IS широкая и плоская THEN класс IS покрытосеменные.

6 IF type IS деревья AND форма листа IS не широкая и плоская THEN класс IS голосеменные.

7 IF стебель IS зеленый THEN type IS травянистый.

8 IF стебель IS древесный AND положение IS стелющееся THEN type IS лианы.

9 IF стебель IS древесный AND положение IS прямостоящее AND один основной ствол IS да THEN type IS деревья.

10 IF стебель IS древесный AND положение IS прямостоящее AND один основной ствол IS нет THEN type IS кустарниковые.

11 IF голова IS болит AND кости IS ломит AND глаза IS слезятся THEN заболевание IS грипп.
